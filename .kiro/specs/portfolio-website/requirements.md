# 需求文档

## 简介

本项目是一个个人作品集网站，用于展示个人简历、技能和项目作品。网站将托管在 GitHub Pages 上，提供给潜在雇主、合作伙伴或其他访问者浏览。

## 术语表

- **作品集网站 (Portfolio Website)**: 展示个人信息、技能和项目作品的静态网站
- **GitHub Pages**: GitHub 提供的免费静态网站托管服务
- **访问者 (Visitor)**: 浏览作品集网站的用户
- **项目卡片 (Project Card)**: 展示单个项目信息的 UI 组件
- **项目类型 (Project Type)**: 项目的分类标签，如 AI-视觉、AI-LLM、Web-IoT、桌面应用等
- **多媒体资源 (Media Resources)**: 项目的展示素材，包括截图、GIF 动图、架构图等
- **国际化 (i18n)**: 支持多语言切换的功能
- **当前语言 (Current Locale)**: 用户当前选择的显示语言（中文或英文）

## 需求

### 需求 1

**用户故事:** 作为访问者，我想查看个人简介信息，以便了解网站主人的背景和专业领域。

#### 验收标准

1. WHEN 访问者打开网站首页 THEN 作品集网站 SHALL 显示个人头像、姓名和职业标题
2. WHEN 访问者浏览简介区域 THEN 作品集网站 SHALL 展示个人简介文字描述（不超过 300 字）
3. WHEN 访问者查看联系方式 THEN 作品集网站 SHALL 显示邮箱和社交媒体链接（GitHub、LinkedIn 等）

### 需求 2

**用户故事:** 作为访问者，我想查看技能列表，以便了解网站主人的专业能力。

#### 验收标准

1. WHEN 访问者浏览技能区域 THEN 作品集网站 SHALL 以分类方式展示技能标签
2. WHEN 技能数据加载完成 THEN 作品集网站 SHALL 按类别（如前端、后端、工具等）分组显示技能
3. WHEN 访问者查看单个技能 THEN 作品集网站 SHALL 显示技能名称和熟练程度指示

### 需求 3

**用户故事:** 作为访问者，我想浏览项目作品，以便评估网站主人的实际工作能力。

#### 验收标准

1. WHEN 访问者进入项目展示区域 THEN 作品集网站 SHALL 以卡片形式展示项目列表
2. WHEN 显示项目卡片 THEN 作品集网站 SHALL 包含项目名称、描述、技术栈和链接
3. WHEN 访问者点击项目链接 THEN 作品集网站 SHALL 在新标签页打开项目详情或源码仓库
4. IF 项目包含预览图 THEN 作品集网站 SHALL 在项目卡片中显示缩略图
5. WHEN 显示项目卡片 THEN 作品集网站 SHALL 显示项目类型标签（如 AI-视觉、AI-LLM、Web-IoT、桌面应用等）
6. IF 项目包含多媒体资源（多张截图、GIF、架构图）THEN 作品集网站 SHALL 支持在卡片中展示或点击查看详情
7. IF 项目无法提供在线演示 THEN 作品集网站 SHALL 通过截图、架构图或 GIF 动图展示项目效果

### 需求 4

**用户故事:** 作为访问者，我想在不同设备上都能良好浏览网站，以便随时随地查看内容。

#### 验收标准

1. WHEN 访问者使用移动设备访问 THEN 作品集网站 SHALL 自适应调整布局以适配屏幕宽度
2. WHEN 屏幕宽度小于 768px THEN 作品集网站 SHALL 切换为单列布局
3. WHEN 屏幕宽度大于等于 768px THEN 作品集网站 SHALL 使用多列网格布局

### 需求 5

**用户故事:** 作为网站主人，我想将网站部署到 GitHub Pages，以便免费托管并分享给他人。

#### 验收标准

1. WHEN 代码推送到 GitHub 仓库 THEN 作品集网站 SHALL 支持通过 GitHub Pages 自动部署
2. WHEN 部署完成 THEN 作品集网站 SHALL 可通过 `username.github.io` 或自定义域名访问
3. WHEN 构建网站 THEN 作品集网站 SHALL 生成纯静态文件（HTML、CSS、JavaScript）

### 需求 6

**用户故事:** 作为访问者，我想有良好的视觉体验，以便愉快地浏览网站内容。

#### 验收标准

1. WHEN 访问者浏览网站 THEN 作品集网站 SHALL 使用蓝白冷色调配色方案（主色调为蓝色系，背景为白色或浅灰色）
2. WHEN 页面加载完成 THEN 作品集网站 SHALL 在 3 秒内完成首屏渲染
3. WHEN 访问者与页面交互 THEN 作品集网站 SHALL 提供平滑的过渡动画效果
4. WHEN 显示强调元素 THEN 作品集网站 SHALL 使用蓝色系作为主要强调色

### 需求 7

**用户故事:** 作为访问者，我想使用我熟悉的语言浏览网站，以便更好地理解内容。

#### 验收标准

1. WHEN 访问者首次访问网站 THEN 作品集网站 SHALL 根据浏览器语言偏好自动选择中文或英文
2. WHEN 访问者点击语言切换按钮 THEN 作品集网站 SHALL 在中文和英文之间切换所有文本内容
3. WHEN 语言切换完成 THEN 作品集网站 SHALL 将语言偏好保存到本地存储
4. WHEN 访问者再次访问网站 THEN 作品集网站 SHALL 恢复上次选择的语言设置
5. WHEN 切换语言 THEN 作品集网站 SHALL 保持页面布局和样式不变，仅更新文本内容
