# 需求文档

## 简介

本功能为作品集网站增加项目详情页，用于充分展示每个项目的 README 文档、演示视频、截图画廊和架构图等多媒体资源。访问者可以从项目卡片点击进入独立的详情页面，深入了解项目的完整信息。

## 术语表

- **项目详情页 (Project Detail Page)**: 展示单个项目完整信息的独立页面
- **README 渲染器 (README Renderer)**: 将 Markdown 格式的 README 文件转换为 HTML 的组件
- **视频播放器 (Video Player)**: 支持播放 MP4 格式演示视频的组件
- **截图画廊 (Screenshot Gallery)**: 以网格或轮播形式展示多张项目截图的组件
- **架构图查看器 (Architecture Viewer)**: 展示项目架构图（PDF 或图片）的组件
- **面包屑导航 (Breadcrumb)**: 显示当前页面层级路径的导航组件
- **项目资源目录 (Project Assets)**: Projects 文件夹下各项目的 README、Images、Video 等资源

## 需求

### 需求 1

**用户故事:** 作为访问者，我想从项目卡片进入详情页，以便查看项目的完整信息。

#### 验收标准

1. WHEN 访问者点击项目卡片 THEN 作品集网站 SHALL 导航到该项目的独立详情页面
2. WHEN 详情页加载完成 THEN 作品集网站 SHALL 显示面包屑导航（首页 > 项目 > 项目名称）
3. WHEN 访问者点击面包屑中的"首页"或"项目" THEN 作品集网站 SHALL 返回对应页面位置
4. WHEN 详情页显示 THEN 作品集网站 SHALL 在页面顶部显示项目标题、类型标签和技术栈

### 需求 2

**用户故事:** 作为访问者，我想查看项目的 README 文档内容，以便了解项目的功能特性和技术细节。

#### 验收标准

1. WHEN 项目包含 README 文件 THEN 作品集网站 SHALL 将 Markdown 内容渲染为格式化的 HTML
2. WHEN 渲染 README 内容 THEN 作品集网站 SHALL 正确显示标题、列表、代码块、表格等 Markdown 元素
3. WHEN README 包含代码块 THEN 作品集网站 SHALL 应用语法高亮样式
4. IF 项目无 README 文件 THEN 作品集网站 SHALL 显示项目的基本描述信息

### 需求 3

**用户故事:** 作为访问者，我想观看项目的演示视频，以便直观了解项目的实际运行效果。

#### 验收标准

1. IF 项目包含演示视频 THEN 作品集网站 SHALL 显示视频播放器组件
2. WHEN 显示视频播放器 THEN 作品集网站 SHALL 支持播放、暂停、进度拖动和音量控制
3. WHEN 项目包含多个视频 THEN 作品集网站 SHALL 以列表形式展示所有视频，支持切换播放
4. WHEN 视频加载失败 THEN 作品集网站 SHALL 显示友好的错误提示信息

### 需求 4

**用户故事:** 作为访问者，我想浏览项目的截图画廊，以便查看项目的界面设计和功能展示。

#### 验收标准

1. IF 项目包含截图 THEN 作品集网站 SHALL 以网格形式展示所有截图缩略图
2. WHEN 访问者点击截图缩略图 THEN 作品集网站 SHALL 打开灯箱模式显示大图
3. WHEN 灯箱模式打开 THEN 作品集网站 SHALL 支持左右切换查看其他截图
4. WHEN 灯箱模式打开 THEN 作品集网站 SHALL 支持点击外部区域或按 ESC 键关闭

### 需求 5

**用户故事:** 作为访问者，我想查看项目的架构图，以便理解项目的技术架构设计。

#### 验收标准

1. IF 项目包含架构图（PDF 或图片）THEN 作品集网站 SHALL 在详情页显示架构图预览
2. WHEN 架构图为 PDF 格式 THEN 作品集网站 SHALL 提供下载链接或在新标签页打开
3. WHEN 架构图为图片格式 THEN 作品集网站 SHALL 支持点击放大查看

### 需求 6

**用户故事:** 作为访问者，我想在详情页也能切换语言，以便用熟悉的语言阅读内容。

#### 验收标准

1. WHEN 访问者在详情页切换语言 THEN 作品集网站 SHALL 更新页面所有可翻译文本
2. WHEN 语言切换 THEN 作品集网站 SHALL 根据当前语言显示对应的 README 文件（如 README_EN.md 或 README.md）
3. WHEN 切换语言后刷新页面 THEN 作品集网站 SHALL 保持用户选择的语言设置

### 需求 7

**用户故事:** 作为访问者，我想在不同设备上都能良好浏览详情页，以便随时随地查看项目信息。

#### 验收标准

1. WHEN 访问者使用移动设备访问详情页 THEN 作品集网站 SHALL 自适应调整布局
2. WHEN 屏幕宽度小于 768px THEN 作品集网站 SHALL 将视频和截图区域改为单列布局
3. WHEN 在移动端查看截图画廊 THEN 作品集网站 SHALL 支持触摸滑动切换图片
